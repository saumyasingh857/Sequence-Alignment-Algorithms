# Sequence Alignment Algorithms

This repository contains Python implementations of two fundamental sequence alignment algorithms used in bioinformatics.

## Algorithms Included

### 1. Needleman-Wunsch Algorithm
- Global Sequence Alignment
- Dynamic Programming
- Finds the best alignment across the entire sequence.

### 2. Smith-Waterman Algorithm
- Local Sequence Alignment
- Dynamic Programming
- Finds the highest-scoring matching region between two sequences.

## Features

- User input sequences
- Alignment score calculation
- Global and local sequence alignment
- Pure Python implementation
- Beginner-friendly code

## Technologies Used

- Python
- Dynamic Programming
- Bioinformatics
